# Nota:

Consumiendo API MapBox y OpenWeatherMaps, y paquetes Axios, Colors e Inquirer

```
--------------------------
         CLIMA-APP        
--------------------------

? Seleccioná una opción (Use arrow keys)
> 1 - Buscar ciudad
  2 - Mostrar historial de busqueda     
  0 - Salir
```
